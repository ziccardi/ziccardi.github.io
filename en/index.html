Nagios NRPE implementations

A set of different NRPE implementations compatible with the original one

JNRPE

A Java NRPE implementation that allows the execution of plugins written with Java without the overhead of a new JVM for each plugin execution

jNRPE

A javascript implemenentation of NRPE that allows to easily execute plugins written in javascript

NRPE++

A C++ implementation of NRPE based on boost ASIO libraries
